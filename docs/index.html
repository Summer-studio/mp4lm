Open model artifacts, made streamable

Exact payloads. Partial fetches. Calm runtime ergonomics.

Mp4LM packages model payloads into a single bundle with a readable manifest, a fast binary index, and chunked regions that are friendly to HTTP range requests. The format keeps canonical bytes exact while leaving room for previews and future runtime hints.

Exact Recovery

Canonical tensor bytes remain byte-exact and independently checksummed.

Selective Load

Chunk tables and offsets make it practical to fetch only what a runtime needs.

Inspectable by Design

Three Planes

A clean split between meaning, bytes, and presentation.

Payload Plane

Tensors, tokenizer assets, adapters, and file artifacts live here as exact, reproducible bytes.

Preview Plane

Optional heatmaps, frames, and summaries can be added without affecting canonical reconstruction.

Control Plane

Header, manifest, index, checksums, and compatibility hints make range-fetch loaders practical.

Quickstart

Small enough to understand, structured enough to grow.

Python

from mp4lm import BundleBuilder, Mp4LMBundle

builder = BundleBuilder(
    model={"name": "toy-green-demo"},
    deterministic=True,
)
builder.add_tensor(
    "layers.0.weight",
    tensor_bytes,
    dtype="float32",
    shape=[2, 2],
)
builder.write("toy_model.mp4lm")

bundle = Mp4LMBundle.open("toy_model.mp4lm")
bundle.verify()
